# 小程序首页优化方案

## 📊 问题诊断

### 当前问题
1. **内容区域空白** - 四个推荐模块没有显示任何内容
2. **无登录入口** - 小程序缺少登录引导
3. **布局不适配** - H5 桌面端布局在小程序上不显示
4. **数据请求可能失败** - 401 错误导致内容加载失败

### 根本原因
- `isDesktop` 判断逻辑在小程序上有问题
- 缺少小程序专属的用户信息展示
- 推荐模块组件未针对小程序优化

## 🔧 技术要点

### 1. 条件编译使用
```vue
<!-- 仅小程序显示 -->
<!-- #ifdef MP-WEIXIN -->
<view>小程序专属内容</view>
<!-- #endif -->

<!-- 仅 H5 显示 -->
<!-- #ifdef H5 -->
<view>H5 专属内容</view>
<!-- #endif -->

<!-- 小程序和 H5 都显示 -->
<view>通用内容</view>
```

### 2. rpx 单位使用
- 小程序使用 `rpx` (responsive pixel)
- 750rpx = 设备宽度
- 1rpx = 0.5px (iPhone 6)

### 3. 小程序登录流程
```javascript
// 1. 调用微信登录
wx.login({
  success: (res) => {
    const code = res.code
    // 2. 发送 code 到后端
    request.post('/auth/wechat/login', { code })
  }
})
```
