### CampusLink首页UI设计详细提示词（基于当前视觉风格+功能重构）

#### 一、设计定位
以“**青春校园+轻量科技感**”为核心风格，延续当前“蓝橙主色+插画元素”的视觉语言，重构首页布局为“顶部聚焦→核心功能→个性化内容→辅助信息”四层结构，同时强化功能引导与校园场景适配。


#### 二、布局与模块设计（PC端优先，兼容H5）
1. **顶部聚焦区（通栏，高度200px）**
   - 背景：渐变蓝（#2F80ED → #409EFF）+ 抽象校园元素（书本、对话气泡）插画；
   - 左侧：品牌标识「CampusLink文字Logo（白色粗体）+ 校园标识（如“XX大学”小字）」；
   - 中间：通栏搜索框（圆角24px，占位符“搜课件/问问题/找活动”，内置热门标签“高数/四六级”，右侧橙色搜索按钮）；
   - 右侧：2个核心CTA按钮（“上传资料赚积分”→白色描边+蓝底，“AI智能答疑”→橙色填充）；
   - 装饰：添加1-2个简约校园插画元素（如举着书本的学生卡通形象）。

2. **核心功能区（网格布局，高度180px）**
   - 布局：2行4列卡片网格，间距20px；
   - 卡片样式：圆角16px，浅色系渐变背景（资料共享→浅蓝、智能问答→浅橙），搭配场景化插画（如“资料共享”=学生抱书本、“互助任务”=人物握手）；
   - 内容：每个卡片包含「功能图标（24px）+ 功能名称（16px粗体）+ 辅助说明（12px浅灰，如“100万+资源”）」；
   - 交互：卡片hover时轻微上浮（2px）+ 阴影加深。

3. **个性化内容区（左右分栏，高度400px）**
   - 左侧（70%宽度）：“为你推荐”模块
     - 标题栏：“为你推荐”（18px粗体）+ 筛选标签（“课件/问答/任务”，橙色激活态）+ “换一批”按钮（橙色文字+刷新图标）；
     - 内容：瀑布流卡片（圆角12px，白色背景+1px浅灰边框），每个卡片包含「类型标签（如“课件”，蓝底白字）+ 标题（16px）+ 简介（14px浅灰）+ 资源信息（上传者/下载量，12px）+ 操作按钮（收藏/下载，图标式）」。
   - 右侧（30%宽度）：“热门榜单”模块
     - 切换标签：“热门问答/精选资料/紧急任务”（橙色下划线激活）；
     - 内容：序号式列表（第1名橙色背景+白字，其余浅灰背景），每条显示「标题（14px）+ 浏览/回答数（12px浅灰）+ 快速操作按钮（“答/抢”，橙色小按钮）」。

4. **辅助信息区（通栏，高度220px）**
   - 布局：3列卡片（积分中心、校园公告、社团动态），间距20px；
   - 积分中心：进度条（蓝橙渐变填充）+ 今日任务列表（图标+任务名+积分值，如“签到+2”）；
   - 校园公告：轻卡片（左侧头像+标题+时间，右侧“重要”红点标签）；
   - 社团动态：横向滚动卡片（活动海报+名称+剩余名额+橙色“报名”按钮）。


#### 三、视觉细节规范
- **色彩**：主色#409EFF（蓝）、辅助色#FF7D00（橙），背景#F5F7FA（浅灰），卡片#FFFFFF；
- **字体**：标题用18-20px粗体，正文14-16px常规体，辅助文字12px浅灰；
- **图标**：统一使用IconPark校园线性图标（24px，主功能填充+描边，次功能仅描边）；
- **插画**：延续当前“扁平卡通+校园场景”风格，用于功能卡片、Banner装饰。


#### 四、多端适配规则
- **PC端**：内容区最大宽度1200px居中，辅助信息区3列布局；
- **H5端**：顶部聚焦区简化为“Logo+搜索+图标”，核心功能区横向滚动，内容区单列布局，按钮尺寸≥44px。
