## 首页 UI 布局与色彩搭配详细开发提示词

### 1. 页面定位

* **目标**：首页是 CampusLink 的信息中枢，展示公告、推荐内容、热门话题及模块入口。设计上需平衡信息密度与操作便捷性，让用户一眼就能感知到系统功能。
* **关键词**：清晰、轻快、结构化、校园氛围。

### 2. Prompt（用于 Figma / Midjourney / UI 生成工具）

> 「设计一款高校学习互助平台的首页界面。顶部包含搜索框与通知图标，中部为模块快捷入口（资源、问答、任务、社团），下方展示热门内容（公告栏、热门问题、热门资源卡片、校园活动轮播）。主色调为校园蓝 (#2E7CF6)，辅色为柔灰与浅白，采用卡片式设计，圆角统一、阴影轻盈，整体布局整洁有层次。页面适配移动端、H5 与小程序端，留白充足，视觉舒适，体现学习与协作氛围。」

### 3. 布局结构

```
┌────────────────────────┐
│ 顶部导航栏：Logo / 搜索框 / 通知图标          │
│────────────────────────│
│ 快捷入口区（4个宫格）：资源 / 问答 / 任务 / 社团 │
│────────────────────────│
│ 公告栏（横向滚动）                          │
│────────────────────────│
│ 热门内容卡片区：                           │
│   - 热门问题卡片（含AI标签）                │
│   - 热门资源卡片（带下载数与标签）          │
│────────────────────────│
│ 校园活动轮播图（横滑）                     │
└────────────────────────┘
```

* 页面采用上下滚动结构，分区清晰。
* 卡片之间间距 12px；页边距左右各 16px。
* 顶部固定导航栏，支持下拉隐藏与上滑显示动画。

### 4. 色彩搭配建议

| 区域     | 元素  | 色值        | 说明           |
| ------ | --- | --------- | ------------ |
| 背景     | 主背景 | `#F7F8FA` | 柔和中性灰，减少视觉疲劳 |
| 顶部栏    | 背景  | `#FFFFFF` | 与页面主体区隔      |
| 主色     | 主蓝  | `#2E7CF6` | 交互与高亮用色      |
| 辅助色    | 学院紫 | `#6C5CE7` | 用于标签或按钮高亮    |
| 文本主色   | 深灰  | `#0F172A` | 标题、正文文字      |
| 次级文本   | 中灰  | `#6B7280` | 时间、标签说明      |
| 边框线    | 浅灰  | `#E5E7EB` | 模块分隔线        |
| 活动卡片背景 | 白色  | `#FFFFFF` | 提升层次与明亮感     |
| 按钮主色   | 蓝   | `#2E7CF6` | 按钮及链接文字统一色调  |

### 5. 字体与层级

| 元素    | 字号(px) | 字重  | 用途          |
| ----- | ------ | --- | ----------- |
| 标题    | 18–20  | 700 | 区块标题“热门资源”等 |
| 副标题   | 14–16  | 500 | 模块说明        |
| 正文    | 14     | 400 | 内容描述、时间、评论数 |
| 标签    | 12     | 500 | 卡片内类别标识     |
| 搜索框占位 | 14     | 400 | 灰色说明文字      |

### 6. 动效与交互提示

* **搜索框**：聚焦后放大至 105%，发光边框 `rgba(46,124,246,0.2)`。
* **宫格入口**：轻微浮动悬停动效，点击后波纹反馈。
* **轮播图**：自动播放，分页点主色蓝，滑动带惯性减速效果。
* **卡片**：点击有轻压动画 + 阴影强化，进入详情页过渡渐隐。
* **公告栏**：滚动速率 3s/条，悬停暂停。

### 8. 组件构成提示词（用于生成工具）

> “设计校园首页组件：顶部导航（logo + 搜索框 + 通知图标）、四宫格模块入口（资源、问答、任务、社团）、公告栏（滚动文字）、热门卡片（问题/资源）、活动轮播图。整体圆角 12，浅灰背景，主色蓝，阴影柔和，信息分区清晰。强调卡片感与可交互性。”

### 9. 背景与氛围元素建议

* 背景渐变：顶部淡蓝 (`#E8F0FE`) → 底部白 (`#FFFFFF`) 渐变，营造通透层次。
* 装饰图形：圆点阵列或抽象几何形，放置于底部或顶部边缘，控制透明度 10–15%。
* 插画元素：学习、讨论、校园景象线稿风格，保持轻量不干扰主内容。

### 10. 最终 Prompt 汇总

> 「生成一个高校学习互助平台的首页界面。采用浅灰与校园蓝配色，顶部导航包含搜索框与通知图标，下方为四宫格功能入口（资源、问答、任务、社团），中部展示公告栏、热门资源/问题卡片、活动轮播。布局整洁、层次分明，卡片圆角 12、阴影柔和。整体风格清新现代、具有校园氛围，适配移动端与小程序端，支持深浅色模式切换。」
