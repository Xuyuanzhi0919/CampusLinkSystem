一、整体设计语言：现代玻璃态 (Glassmorphism) + 企业级内容平台

轻盈通透：玻璃态磨砂效果，营造空间层次感

呼吸感留白：48px 模块间距，基于 8pt 设计系统

五、交互设计细节
1. 鼠标视差滚动效果（Hero Section）

// Hero 元素根据鼠标位置产生不同速度的移动
.floating-stats .bubble-1 { 
  transform: translate(
    calc(var(--mouse-x) * 8px),   // 浅层移动慢
    calc(var(--mouse-y) * 8px)
  );

.bubble-3 { 
  transform: translate(
    calc(var(--mouse-x) * 20px),  // 深层移动快

  );

// 背景装饰元素反向移动
.deco-block {

    calc(var(--mouse-x) * -12px),  // 负值 = 反向
    calc(var(--mouse-y) * -12px)
  ); 
}

3. 打字机效果（Hero 副标题）
javascript
复制
// HeroBrand.vue 中实现
const displayedText = ref('')
const targetText = '课件共享·学业问答·任务互助·社团活动'

onMounted(() => {
  let index = 0
  const interval = setInterval(() => {
    if (index < targetText.length) {
      displayedText.value += targetText[index]
      index++
    } else clearInterval(interval)
  }, 80)  // 80ms 打字速度
})
