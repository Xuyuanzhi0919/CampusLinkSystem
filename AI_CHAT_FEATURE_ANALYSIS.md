```typescript
interface ChatSession {

  title: string             // 会话标题(自动生成)

  createdAt: number         // 创建时间戳
```

### 6. 性能优化 (95%)

✅ **已实现的优化**

   - 前端: 只发送最近 20 条消息
   - 后端: 进一步限制到最近 10 轮对话
   - **效果**: 节省 60-80% Token 成本

2. **请求超时优化**
   - AI 接口专用超时: 60 秒(默认 10 秒)
   - 防止长时间推理导致超时

3. **消息配对算法**
   - 避免重复配对
   - 支持连续用户消息
   - 处理未回复消息

4. **DOM 操作优化**
   - 使用 `nextTick()` 确保 DOM 更新
   - 代码块复制按钮复用检测
   - 防止重复添加按钮

5. **响应式更新**
   - 使用 `Array.splice()` 触发 Vue 响应式
   - 避免直接修改数组元素

6. **打字机效果**
   - 逐字显示 AI 回复
   - 流畅的视觉体验
   - 50ms 延迟(可调整)

## 🐛 已修复的问题

### 问题 1: 消息配对逻辑 Bug
**症状**: 连续用户消息会错误配对
**修复**: 重写配对算法,使用 `processedIndices` 跟踪

### 问题 2: 打字机效果无响应
**症状**: AI 回复不显示逐字效果,刷新后才出现
**修复**: 使用 `Array.splice()` 替代直接赋值,改用 `<view>` 替代 `<text>`

### 问题 3: markdown-it-emoji 导入失败
**症状**: `SyntaxError: does not provide an export named 'default'`
**修复**: 使用命名导入 `import { full as markdownItEmoji } from 'markdown-it-emoji'`

### 问题 4: API 超时
**症状**: 后端返回 200 OK,但前端显示超时
**修复**: AI 接口超时时间从 10 秒增加到 60 秒

### 问题 5: 复制按钮不显示
**症状**: 代码块没有复制按钮
**修复**: 修改 DOM 获取方式,使用 `ref` + `$el` + 降级方案

### 问题 6: 会话列表项超出容器宽度
**症状**: 会话项宽度超出抽屉容器
**修复**: 对齐搜索栏的 margin 模型,统一使用 `margin: 0 12px`

### 问题 7: 删除/重命名模态框被遮挡
**症状**: 模态框隐藏在抽屉后面
**修复**: 操作前自动关闭抽屉,延迟 350ms 后显示模态框
