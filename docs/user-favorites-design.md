# 我的收藏页面设计文档

## 📋 页面概述

**页面路径**: `/pages/user/favorites`
**功能描述**: 展示用户收藏的资源、问题、任务列表,支持筛选、取消收藏、查看详情
**入口来源**: 用户中心主页 "我的收藏" 功能入口

---

## 🔧 功能需求

### 1. 类型筛选

**Tab 选项**:
- 全部 (默认)
- 资源 (resource)
- 问答 (question)
- 任务 (task)

**交互逻辑**:
- 点击 Tab 切换类型
- 切换时重置分页,从第1页开始加载
- 当前选中 Tab 高亮显示

### 2. 收藏卡片

**卡片信息**:
| 字段 | 说明 | 示例 |
|------|------|------|
| 类型标签 | 收藏对象类型 | [资源] [问答] [任务] |
| 标题 | 收藏对象标题 | 高等数学期末复习资料 |
| 创建者 | 发布者昵称 | 👤 张三 |
| 浏览数 | 查看次数 | 👁 123 |
| 点赞数 | 点赞数量 | ❤️ 45 |
| 收藏时间 | 用户收藏的时间 | 收藏于 2024-01-15 |

**卡片操作**:
- **点击卡片**: 跳转到对应详情页
  - 资源 → `/pages/resource/detail?id=xxx`
  - 问答 → `/pages/question/detail?id=xxx`
  - 任务 → `/pages/task/detail?id=xxx`
- **长按卡片**: 显示操作菜单
  - 取消收藏
  - 复制标题
  - 分享

### 3. 取消收藏

**操作方式**:
1. 长按卡片弹出操作菜单
2. 点击 "取消收藏"
3. 显示确认对话框
4. 确认后调用 API 删除收藏
5. 从列表中移除该项

**API**: `DELETE /favorite/{targetType}/{targetId}`

### 4. 空状态

**显示条件**:
- 列表为空时显示

**内容**:
```
     😔
  暂无收藏内容
去收藏喜欢的资源、问题或任务吧~
```

### 5. 分页加载

**加载方式**: 下拉刷新 + 上拉加载更多

**下拉刷新**:
- 重置分页到第1页
- 重新加载数据
- 显示刷新动画

**上拉加载更多**:
- 滚动到底部触发
- 显示 "加载中..." 提示
- 加载下一页数据
- 如果没有更多数据,显示 "没有更多了"

## 🎯 交互细节

### 1. 类型标签

**样式**:
- 资源: 绿色背景 `#D1FAE5` + 绿色文字 `#10B981`
- 问答: 橙色背景 `#FEF3C7` + 橙色文字 `#F59E0B`
- 任务: 紫色背景 `#EDE9FE` + 紫色文字 `#8B5CF6`
- 圆角 `4rpx`,内边距 `8rpx 16rpx`

### 2. 卡片布局

**结构**:
```
┌─────────────────────────────────┐
│ [资源]  高等数学期末复习资料... │ ← 类型标签 + 标题
│ 包含历年真题和知识点总结...     │ ← 描述(灰色,最多2行)
│ 👤 张三 • 👁 123 • ❤️ 45        │ ← 元信息
│ 收藏于 2024-01-15 12:30        │ ← 收藏时间(浅灰)
└─────────────────────────────────┘
```

**间距**:
- 卡片间距: `20rpx`
- 内边距: `32rpx`
- 行间距: `16rpx`

### 3. 时间显示

**格式化规则**:
- 今天: "今天 12:30"
- 昨天: "昨天 12:30"
- 本年: "01-15 12:30"
- 往年: "2023-01-15"

### 4. 长按菜单

**弹出方式**: uni.showActionSheet

**菜单项**:
```javascript
[
  '取消收藏',
  '复制标题',
  '分享',
  '取消'
]
```

## 🔌 API 对接

### 1. 获取收藏列表

**接口**: `GET /favorite/my`

**请求参数**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| targetType | String | 否 | 类型筛选 (resource/question/task) |
| page | Integer | 否 | 页码,默认1 |
| pageSize | Integer | 否 | 每页数量,默认20 |

**响应示例**:
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "list": [
      {
        "favoriteId": 1,
        "targetType": "resource",
        "targetId": 123,
        "title": "高等数学期末复习资料",
        "description": "包含历年真题和知识点总结",
        "creatorId": 456,
        "creatorName": "张三",
        "viewCount": 123,
        "likeCount": 45,
        "createdAt": "2024-01-15T12:30:00",
        "targetCreatedAt": "2024-01-10T10:00:00"
      }
    ],
    "total": 50,
    "page": 1,
    "pageSize": 20,
    "totalPages": 3
  }
}
```

### 2. 取消收藏

**接口**: `DELETE /favorite/{targetType}/{targetId}`

**路径参数**:
| 参数 | 类型 | 说明 |
|------|------|------|
| targetType | String | 对象类型 |
| targetId | Long | 对象ID |

**响应示例**:
```json
{
  "code": 200,
  "message": "取消收藏成功",
  "data": null
}
```

## 📝 注意事项

1. **性能优化**
   - 使用虚拟列表优化长列表渲染
   - 图片懒加载
   - 防抖搜索

2. **用户体验**
   - 取消收藏需要二次确认
   - 操作成功后给予反馈提示
   - 网络异常时显示重试按钮

3. **异常处理**
   - 被收藏对象已删除时的处理
   - 网络超时重试机制
   - 空数据状态提示

4. **数据同步**
   - 取消收藏后更新列表
   - 从详情页返回时刷新列表
   - 保持与服务端数据一致
