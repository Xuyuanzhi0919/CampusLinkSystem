# 【我的】页面重设计文档

## 📋 设计理念

从**功能展示**转向**行为引导** + **成长激励**

### 核心原则

1. **身份 + 进度感** - "我在 CampusLink 走到哪了"
2. **行为优先** - 告诉用户下一步该干什么,而不是我有什么功能
3. **轻管理、重行动** - 设置类内容必须降权

---

## 🎨 页面结构（三段式）

### ① 顶部：身份卡（Identity Card）

**位置**: `UserProfileHeader.vue:3-56`

**结构**:
```
┌──────────────────────────────────┐
│  🧑 Yuan0919   Lv.10   ⭐          │
│     软件工程 · XX大学              │
│                                  │
│  ┌────────────────────────────┐ │
│  │ 积分 9758  |  排名 Top 12%  │ │
│  └────────────────────────────┘ │
│                                  │
│  [每日签到]     [发布内容]       │
└──────────────────────────────────┘
```

**关键特性**:
- 蓝色渐变背景（品牌色 #2563EB → #1D4ED8）
- 装饰性光晕背景（半透明圆形）
- 成就数据行（积分橙色、排名青色）
- 毛玻璃效果（backdrop-filter: blur(10px)）
- **核心行为按钮**:
  - 每日签到：白色轮廓按钮
  - 发布内容：橙色渐变按钮（强转化）

**行为设计**:
- 点击"发布内容"弹出选项：发布资源/任务/提问/活动
- 点击成就数据行跳转积分详情
- 点击头像预览大图

---

### ② 中部：可操作化数据卡

**位置**: `UserProfileHeader.vue:59-77`

**布局**: 2x2 网格

```
┌─────────────┬─────────────┐
│ 📄 我的资源  │ 💬 我的回答  │
│    5        │    17       │
│ 查看 →      │ 查看 →      │
├─────────────┼─────────────┤
│ ⏰ 进行中任务│ ❤️  我的收藏 │
│    2        │    11       │
│ 去完成 →    │ 查看 →      │
└─────────────┴─────────────┘
```

**关键创新**:
- 每个数字都是可点击的入口
- 带图标 + 颜色区分（蓝/绿/橙/红）
- **行为型文案**：
  - 有进行中任务 → "去完成"（橙色高亮边框）
  - 其他 → "查看"
- 点击时浮起效果（translateY(-2rpx)）

**数据来源**:
```typescript
{
  resources: stats?.resourceCount,
  questions: stats?.questionCount,
  tasks: stats?.taskAcceptedCount, // 进行中的任务
  favorites: stats?.favoriteCount
}
```

### ③ 功能入口分组

**位置**: `FunctionGrid.vue`

#### A. 我的内容（最高优先级）

**图标**: 📁 folder（蓝色）

- 我的资源（file-text）
- 我的回答（message-circle）
- 我的活动（calendar）
- 我的社团（users）

#### B. 我的互动

**图标**: 💬 message-square（绿色）

- 通知中心（bell） - 未读角标
- 我的消息（mail） - **红色高亮角标**
- 收藏 / 点赞（heart）

#### C. 我的成长

**图标**: 📈 trending-up（橙色）

- 积分记录（award）
- 成就徽章（star）- 待开发
- 等级说明（info）- 待开发

#### D. 设置与管理（降权）

**图标**: ⚙️ settings（灰色）

- 编辑资料（user）
- 修改密码（lock）
- 系统设置（sliders）
- 关于 CampusLink（info）

**视觉降权手段**:
- 阴影更浅（0 2rpx 8rpx）
- 图标颜色灰度（$gray-500）
- 标签字重更轻（font-weight: 400）

---

### ④ 底部：退出登录单独隔离

**位置**: `AccountActions.vue`

**样式**:
```
┌──────────────────────────────────┐
│                                  │
│  [🚪 退出登录]                   │
│  (红色边框,中心对齐,单独一行)      │
│                                  │
└──────────────────────────────────┘
```

**特点**:
- 红色边框（border: 2rpx solid $error-100）
- 二次确认弹窗
- 与功能区保持间距（margin-top: 16rpx）
