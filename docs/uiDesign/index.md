二、核心功能区：场景化插画 + 层级区分
当前问题：功能卡片色彩杂乱、图标识别性弱、无场景关联对标目标页优化：
卡片风格统一：
功能卡片改为浅色系渐变背景 + 场景化插画（如 “资料共享”= 学生抱书本、“智能问答”= 学生喝奶茶看手机），替代当前纯色卡片；
核心功能（资料共享 / 智能问答）用品牌蓝渐变，次要功能（附近同学 / 积分商城）用浅灰 + 品牌色点缀，减少高饱和非品牌色使用。
信息层级强化：
每个卡片增加 “功能图标（24px 线性风格）+ 功能名称（16px 粗体）+ 辅助说明（12px 浅灰，如‘100 万 + 资源’）”，替代当前单一文字；
卡片 hover 时增加 **“上浮 2px + 阴影加深” 交互 **，提升精致度。
三、内容推荐区：瀑布流 + 视觉分层
当前问题：推荐卡片排版拥挤、信息冗余、无视觉重点对标目标页优化：
布局与样式：
改为瀑布流卡片布局（卡片高度随内容自适应），统一使用 “8px 圆角 + 1px 浅灰边框 + 轻微阴影”，替代当前紧凑的网格；
卡片增加类型标签（如 “课件”= 蓝底白字、“问答”= 橙底白字），放在标题左侧，快速区分内容类型。
信息精简：
卡片内容简化为 “标题（16px 粗体）+ 简介（14px 浅灰）+ 核心操作按钮（收藏 / 下载）”，点赞 / 浏览数移至卡片右下角（12px 浅灰）；
“换一批” 按钮改为橙色填充 + 刷新图标，放在标签栏右侧，点击后增加 “卡片渐显” 动画。
四、辅助模块：场景化 + 动态交互
当前问题：积分 / 公告 / 社团模块样式单调、无场景关联对标目标页优化：
积分中心：
进度条改为蓝橙渐变分段式填充（每 200 积分一个节点），增加 “积分可兑换” 提示（如 “可兑打印券”）；
今日任务项前增加场景图标（签到 = 日历、邀请 = 好友），任务右侧用 “+2” 替代当前的绿色对勾，强化激励感知。
校园公告：
改为 “轻卡片 + 头像标识” 样式，增加紧急程度标签（重要 = 红色、普通 = 灰色），未读公告加红点；
社团动态：
卡片增加 **“剩余名额进度条”**（如 “28/50 人”），报名按钮改为橙色填充，hover 时显示 “立即报名” 提示。
五、全局规范统一
色彩：主色 #409EFF、辅助色 #FF7D00，中性色按 “深灰 / 中灰 / 浅灰” 分层，减少非品牌色；
字体：标题 18-24px 粗体、正文 14-16px 常规体、辅助文字 12px 浅灰，统一使用无衬线字体；
交互：所有可点击元素 hover 时 “上浮 + 阴影”，操作时增加加载动画，提升反馈感。
