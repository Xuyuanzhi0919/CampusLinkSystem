# 4) 登录弹窗（Tab：**登录**）

## 字段

1. **账户**（邮箱 / 手机号，自适应识别）

   * 占位：`输入邮箱或手机号`
2. **密码**

   * 占位：`输入密码`
   * 右侧眼睛切换显隐；支持 CapsLock 提示。
3. **验证码**（可选，风控触发/失败超过 3 次出现）

   * 格式：图形验证码或短信验证码。

## 辅助

* 记住我（7 天）：复选框；
* 忘记密码：右侧链接，进入找回流程。
* 安全提示：输入框下方轻提示：“请勿在公共设备勾选记住密码”。

## 主按钮

* 文案：**登录**
* 交互：加载时按钮变为 loading（圆环 16px）；禁用态 40% 透明度；
* 成功动效：按钮短促填充 → 勾号 → Modal 渐隐（300ms）。

## 错误与校验（示例文案）

* 帐号为空：`请输入邮箱或手机号`
* 格式错误：`格式不正确，请检查输入`
* 密码为空：`请输入密码`
* 账号或密码错误：`账号或密码错误，已为你重置输入焦点`
* 频繁失败：显示验证码 + 提示 `为了你的账户安全，请完成验证码`

## 三方登录（可选）

* **微信 / QQ / Apple / GitHub**（按实际可用选择 2–3 个）：

  * 图标按钮（40×40，描边按钮，Hover 填充主色）；
  * 放在下方“其他方式登录”分隔线上方。

---

# 5) 注册弹窗（Tab：**注册**）

## 字段

1. **昵称**（2–16 字，禁特殊字符）
2. **邮箱 / 手机号**（二选一，建议默认邮箱）
3. **验证码**

   * 邮箱验证码 or 短信验证码；按钮 `获取验证码`（60s 倒计时）。
4. **密码**（≥8 位，含大小写/数字/符号至少两类）
5. **确认密码**（需与上方一致）
6. **同意协议**（必勾）：`《用户协议》` `《隐私政策》`

## 主按钮

* 文案：**注册并登录**（若需要邮箱激活改为“注册并去激活”）
* 成功态：弹出“已发送验证邮件”条幅，提供**重新发送**与**打开邮箱**按钮。

## 校验文案示例

* 昵称不合法：`昵称应为2-16个字符，不含特殊符号`
* 邮箱错误：`邮箱格式不正确`
* 验证码：`请输入 6 位验证码`
* 密码过弱：`建议至少 8 位且包含字母与数字`
* 两次密码不一致：`两次输入的密码不一致`

# 6) 视觉细节（控件与按钮）

* **输入框** 高 44，圆角 10；描边默认 `#E5E7EB`，聚焦 `#2563EB` + 外发光 `0 0 0 3px rgba(37,99,235,.12)`；
* **主按钮** 高 44，圆角 12；

  * 默认：主色 `#2563EB`；
  * Hover：亮度 +6%；
  * Press：下沉 1px；
* **次按钮/描边按钮**：描边 `#CBD5E1`，文字主色；
* **Tab**：高 36，胶囊；激活填充主色，未激活描边；切换 200ms 滑块动效。

---

# 7) 动效与状态

* Modal 入场：**Scale 0.98 → 1** + 淡入（200ms）
* Tab 切换：内容区域**交叉淡入**（150ms）
* 错误态：输入框描边转为 `#DC2626`，下方出现红色说明，Icon 抖动 1 次（轻微）。
* 获取验证码：倒计时数字动画（mm:ss），结束自动恢复按钮。

---

# 8) 无障碍 & 可用性

* 所有控件可键盘访问（Tab/Shift+Tab/Enter/Esc）；
* 屏幕阅读器 label 完整（`aria-label`）；
* 焦点环 2px 主色 35% 透明；
* 错误信息与输入框通过 `aria-describedby` 关联。

---

# 9) 安全策略（推荐）

* 登录失败 5 次 → 验证码；
* 注册/找回短信/邮件验证码单 IP 限流（60s + 每小时上限）；
* 密码不回显、粘贴可选；
* 第三方登录需补充手机号/邮箱完成绑定；
* 异地登录提醒（可选）。
